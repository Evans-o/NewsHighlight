<h2>Description</h2>

<h2>Set-up and Installation</h2>
- Clone the Repo
- Edit the start.sh file with your api key from the news.org website

<h3>Live link</h3>
 view the project: https://evans-o.github.io/NewsHighlight/
- Install python 3.6
- Run chmod a+x run.py
- Run ./run.py
